CODE 7 & ESCOUADE

ALIGNEMENT // DISCIPLINE

FORMATION // RASSEMBLEMENT_PROTOCOLE
► CHARGEMENT: CODE_7 / CODE_10 / PTMENU

ALIGNEMENT ET DISCIPLINE

PRÉSENTATION

But : Maintenir une présentation irréprochable lors des rassemblements ou formations.

Procédure :

À l'ordre "Code 7" ou "Formation", toutes les unités :

  • ► S'alignent dans la colonne par rapport à son grade.
  • ► Maintiennent une distance régulière.
  • ► Regardent le sol lors des Code 7 / Pour les formations à voir en fonction du grade.
  • ► Silence total lors du code 7 et formation. (Si besoin de parler effectuer une 10-75)

CODE 7

RASSEMBLEMENT NEXUS
  • ►Toutes les unités disponibles se regroupent à un point désigné.
  • ►Lors de l'arrivée sur place → Salut militaire.
  • ►Attendre l'ordre de Repos.
  • ►Ordre hiérarchique : Plus haut gradés placé à gauche, bas gradé à droite.

CODE 10

RASSEMBLEMENT NEXUS → ANCIENNE SALLE CODE 7
  • ►Toutes les unités disponibles se regroupent à un point désigné.
  • ►Lors de l'arrivée sur place → Salut militaire ! (Uniquement si SqL ou plus qui fait le code 10 !)
  • ►Attendre l'ordre de Repos.
  • ►Ordre hiérarchique : Plus haut gradés placé à gauche, bas gradé à droite.
