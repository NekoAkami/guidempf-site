PROCEDURE DE CONSCRIPTION

SOP-03 // AUTORISATION REQUISE : 03+

La conscription est le processus de recrutement de nouveaux agents dans les rangs de la Metropolice Force. Seuls les officiers de grade 03+ sont habilites a initier ce processus.

CRITERES DE SELECTION

PRE-REQUIS CANDIDAT
REQUIS
  • + Citoyen avec CID valide
  • + Aucun verdict actif supérieur a V2
  • + Comportement cooperatif observe
  • + Capacite physique adequate
DISQUALIFIANT
  • - Statut anti-citoyen actuel ou passe
  • - Historique de contrebande CnB2+
  • - Liens connus avec la resistance
  • - Refus anterieur de cooperer avec la MPF

ETAPES DE CONSCRIPTION

PROCESSUS COMPLET
  1. 01.
    Identification du candidat
    L'officier repere un citoyen potentiel lors de patrouilles. Scanner le CID pour vérifier le casier.
  2. 02.
    Entretien preliminaire
    Conduire le candidat au Nexus. Poser les questions standard : motivation, loyaute, disposition a servir l'Union.
  3. 03.
    Test d'aptitude
    Evaluation physique basique : obéissance aux ordres, reaction au stress, condition physique. Pas de test ecrit.
  4. 04.
    Validation commandement
    Le rapport de conscription est soumis au commandement (05+ ou DvL). Le commandement approuve ou refuse.
  5. 05.
    Integration
    Si approuve : attribution du grade RCT, equipement de base, assignation a un superviseur 03+ pour formation.
  6. 06.
    Cours RCT obligatoire
    La recrue doit suivre le cours RCT dans les 48h suivant l'integration. Echec = retour au statut civil.
