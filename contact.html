SIGNALEMENT DE PROBLEMES

RPT-BUG

Pour signaler un dysfonctionnement, une erreur dans la documentation ou un probleme d'acces, utilisez le canal suivant :

DEPOT GITHUB
Ouvrir une Issue sur le depot du projet avec une description du probleme rencontre.

CONTRIBUTION

CONTRIB

Les contributions au portail de documentation sont soumises a approbation hierarchique. Procedure :

01
Soumettre une proposition via Discord ou GitHub Pull Request
02
Validation par un 05+ ou un Division Leader
03
Integration au portail apres revision et tests

DEMANDE D'ACCES

ACC-REQ

L'acces au portail est reserve au personnel MPF actif. Pour obtenir un acces :

NOUVEL ACCES Demander a un 04+ d'enregistrer votre compte via le panneau d'administration.
ACCES PERDU Contacter un administrateur via Discord pour reinitialisation.
ACCES ADMIN Reserve au Commandement. Attribution directe par un CMD.
