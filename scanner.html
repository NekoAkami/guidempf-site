DISPOSITIF SCANNER

EQUIPEMENT // IDENTIFICATION CIVILE

Le scanner est l'outil d'identification principal de la MPF. Il permet de vérifier le CID d'un citoyen, consulter son casier et valider son statut social.

UTILISATION DU SCANNER

PROCEDURE STANDARD
  1. 01.
    Ordonner l'arret
    "Citoyen, halte. Controle d'identite. Presentez votre CID."
  2. 02.
    Activer le scanner
    Pointer le dispositif vers le citoyen. Le scan s'effectue automatiquement.
  3. 03.
    Lecture des résultats
    Le scanner affiche : nom, CID, statut de loyaute, verdicts actifs, contrebande detectee.
  4. 04.
    Decision
    Selon les résultats : liberation, avertissement verbal, verdict ou incarceration.

INTERPRÉTATION

GUIDE DE REPONSE
RESULTAT ACTION REQUISE
CID VALIDE - AUCUN VERDICT Libérer le citoyen. "Circulation autorisee."
VERDICTS ACTIFS (V1-V3) Avertir le citoyen. Surveillance accrue. Considération de fouille.
VERDICTS ACTIFS (V4+) Fouille obligatoire. Escorte au Nexus pour evaluation. Informer dispatch.
ANTI-CITOYEN DETECTE Arrestation immediate. Aucune negociation. Incarceration directe.
CID INVALIDE / ABSENT Detention pour verification. Possible anti-citoyen ou CID contrefait. V2 minimum.
CONTREBANDE DETECTEE Saisie immediate. Verdict selon classification CnB. Voir table CnB.
