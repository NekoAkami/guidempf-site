Metropolice Force Documentation

COURS THEORIQUES

FORMATION // PROGRAMME EDUCATIF MPF

Programme de formation obligatoire pour toute unité MPF. Chaque cours correspond a un palier de grade et doit etre valide avant toute promotion.

CRS-RCT

COURS RECRUE (RCT)

PREREQUIS : AUCUN

Formation initiale obligatoire. Couvre la doctrine Combine, l'equipement de base, les codes radio, les missions standard, les verdicts et la hierarchie.

6 MODULES // DUREE : 45 MIN // EXAMEN REQUIS
CRS-03

COURS OFFICIER (03+)

PREREQUIS : GRADE 04 MINIMUM

Formation avancee de commandement. Couvre la tactique, les verdicts V7/amputation, la conscription, les inspections et le leadership.

5 MODULES // DUREE : 60 MIN // EXAMEN REQUIS
CRS-05

COURS COMMANDEMENT (05+)

PREREQUIS : GRADE 05 MINIMUM

Formation de haut commandement. Couvre les responsabilites avancees, commandes techniques, incarceration, divisions et privilèges de rang.

5 MODULES // DUREE : 60 MIN // EXAMEN REQUIS
DIRECTIVE DE FORMATION
  • Chaque cours doit etre suivi dans l'ordre hiérarchique (RCT avant 03, 03 avant 05).
  • Les examens sont administres par un officier de grade 03 ou superieur.
  • Un echec a l'examen entraine un delai de 48h avant une seconde tentative.
  • Le commandement peut ordonner un recyclage de formation a tout moment.
