BRECHES ET SANCTIONS INTERNES

GDE-08 // CODE DISCIPLINAIRE

Les breches sont des violations du code de conduite interne. Elles sont traitees par le commandement et peuvent mener a la rétrogradation ou l'amputation de l'unité fautive.

BRECHES MINEURES

SANCTION : AVERTISSEMENT / -1 GRADE
CODE INFRACTION
BRC-M01 Non-respect du code vestimentaire (masque retire, equipement manquant)
BRC-M02 Retard a la prise de poste ou absence non declaree (< 3 jours)
BRC-M03 Communication radio non réglementaire (langage familier, codes incorrects)
BRC-M04 Patrouille solo sans autorisation (01-02 sans superviseur)
BRC-M05 Verdict prononce sans compétence de grade

BRECHES MAJEURES

SANCTION : RETROGRADATION / SUSPENSION
CODE INFRACTION
BRC-J01 Insubordination (refus d'obeir a un ordre direct d'un supérieur)
BRC-J02 Usage excessif de la force (violence non justifiee envers un citoyen)
BRC-J03 Absence non declaree prolongee (> 7 jours)
BRC-J04 Négligence grave entrainant la fuite d'un suspect ou perte de materiel
BRC-J05 Communication d'informations classifiees a des civils

BRECHES CRITIQUES

SANCTION : AMPUTATION IMMEDIATE
CODE INFRACTION
BRC-C01 Trahison envers l'Union Universelle (collaboration avec la resistance)
BRC-C02 Possession ou trafic de contrebande CnB2/CnB3
BRC-C03 Meurtre d'un citoyen ou unité MPF sans autorisation
BRC-C04 Desertion (abandon de poste definitif)
BRC-C05 Corruption (accepter des pots-de-vin, favoriser un citoyen)

PROCESSUS DISCIPLINAIRE

ETAPES
  1. 01.
    Signalement
    La breche est signalee par un temoin ou un supérieur au commandement.
  2. 02.
    Enquete interne
    Un officier 05+ examine les preuves et interroge les parties concernees.
  3. 03.
    Decision
    Le commandement prononce la sanction appropriee selon la gravite.
  4. 04.
    Application
    La sanction est appliquee immediatement. Aucun appel n'est possible.
